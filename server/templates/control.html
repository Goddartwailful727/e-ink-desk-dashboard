PKU Secure 配网说明

1. 新固件启动后会在墨水屏显示配网页信息。

2. 手机连接设备热点 PKU-Ink-Setup-020F3C。

3. 打开 http://192.168.4.1，输入校园网账号和密码。

4. ESP32-S3 会自己用 802.1X 连接 PKU Secure。

5. 联网成功后继续拉取 /dashboard/current.raw 刷新墨水屏。

公网控制台只展示配网说明，不接收校园网密码。密码只填写到 ESP32-S3 自己开的本地页面里；如果要重新配网，按住 BOOT 键重启 3 秒会清除设备上的 Wi-Fi 配置。
